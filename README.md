# Pokedex

An app that displays pokemons and information about them

## Technology
- React.js
- Typescript
- Sass
- Redux

## Setup
- clone repository and in the root directory run 
```bash
npm install
```

## Run in Development
- to start in development run
```bash
npm run start
```
